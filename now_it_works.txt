Модуль 1
  Телеграмм-бот
  Прием команд
    Доступные команды
      add <domain> [domain]...
        Получение файла со списком доменов (pull)
        Проверка на наличие домена
        Добавить домен в файл
        Отправка файла со списком доменов (commit, push)
        Команда для ботов на применение изменений (import url)
      del <domain>
        Получение файла со списком доменов
        Проверка на наличие домена
        Удалить домен из файла
        Коммит
        Команда для ботов на применение изменений (import url)
      refresh
        Команда для ботов на применение изменений (import url)

Модуль 2
  На роутере
  Команда <import> <url>
    Скачивает файл 
    Сравнивает с текущим списком 
    Новое добавляет: kvas add
    Отсутствующее удаляет: kvas del
    Отправляет результат
    Сохраняет текущий список
